== Description ==

<p>By integrating our system with major social media platforms we make it easy for users to share, bookmark and tag content.</p>

<p>The data is used to find people with similar interests who will want that same content.</p>

<p>If someone visits a website and likes an article, not only will it be publicly posted on their stream, it will also be emailed directly to users who have been matched with similar interests.</p>

<p>For more information check out <a href="http://www.sonicblink.com/">http://www.sonicblink.com/</a>

== Installation ==
1. Download the plugin 
3. Unzip the file and place the Facebook Like plugin folder in your WP-Content -> Plugins Folder
4. Log in to your WordPress dashboard.
5. Go to Plugins -> Installed
6. Activate the plugin
7. (optional) Go to The SonicBlink options panel, configure the settings and save

== Frequently Asked Questions ==

= How do you adjust the settings =

After activating the plugin a new menu item will be added under Settings labeled 'SonicBlink'

= My users haven't gotten any e-mails yet =

This is a very early release and we are still finishing the e-mail recommendation system. SonicBlink was released early so that people could get a head start on visitor feedback and have a huge spike in traffic on the first day we launch the e-mail recommendations.

= What happens with all of my old tags =

We integrate them right into the system. However, like all tags, you get more credit when a users up votes your tags.

= Why doesn't it do x, y, or z =

Please let us know what x, y, or z are. http://www.sonicblink.com/
